01 — Industrial Project · 5 months · Full-Stack + AI

AI-Powered Shift Planning Platform

ADFC SPA · Algerian Drilling Fluids Services · Oil & Gas

Full-stack web application automating workforce scheduling for ADFC's 4x4 and 6x3 oil field rotation systems. The scheduling engine uses a Genetic Algorithm to satisfy complex hard constraints (room capacity, team composition, zero conflicts) while optimising soft constraints (fair workload, team stability). Built with React + Flask + PostgreSQL via Supabase.

Full Project ↗
Algorithm Design

Each chromosome encodes a 9-week schedule per employee (team, shift type, room). The fitness function maximises soft constraint scores and penalises hard constraint violations. The GA evolves populations toward feasible, fair schedules validated on real ADFC operational data.

Hard Constraints Enforced
✦ One room per employee — max 72 rooms total
✦ 5 rooms always reserved, never allocated
✦ Fixed team: 1 supervisor + 4 members
✦ Zero room or shift conflicts allowed
✦ Role-based access: employee / HR / supervisor
